Overview of C Language

C is a structured programming language developed by 
Dennis Ritchie in 1973 at Bell Laboratories. It is one of 
the most popular computer languages today because of its 
structure, high-level abstraction, machine independent 
feature etc. C language was developed to write the UNIX 
operating system, hence it is strongly associated with 
UNIX, which is one of the most popular network
 operating system in use today and heart of internet data 
 superhighway.
